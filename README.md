This project is a **Proof of Concept (PoC)** born from the need to bridge the gap between academic/scientific prestige and digital portability. 

Originally designed for the **Aura CIAN** and **El Pez Volador** ecosystems, this system allows researchers and professionals to display their milestones (such as INSAR Awards, Google Cloud, Adobe, or NASA Space Apps certifications) directly on their professional profiles using WordPress, ACF, and BuddyPress.
